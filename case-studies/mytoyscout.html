My Toy Scout

Affiliate marketing site built to review, rank, and monetize children’s toys

Problem

I wanted to create a site that showcases hand-picked, SEO-optimized toy reviews that convert affiliate traffic into income while providing value to parents and gift shoppers.

Solution

Using React and custom routing, I developed a fully responsive site that features toy listings with long-form reviews, Amazon affiliate links, blog posts, mascot branding, and social-ready layout.

Result

MyToyScout.com now drives daily traffic from search engines and Pinterest. It offers curated toy lists for categories like Outdoor, STEM, and Educational — all monetized through affiliate links.

Technologies Used

  • React
  • React Router
  • Netlify Hosting
  • Amazon Affiliate Integration
  • SEO Optimization
